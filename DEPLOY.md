### 1️⃣ Создай пустой репо на GitHub
- Заходи https://github.com/new
- Name: `digital-clinic`
- Description: `Digital Clinic mini-app for Freedom SuperApp Kazakhstan`
- Public
- **Create repository** (без README, .gitignore, license)

### 2️⃣ Запусти в terminal:
```bash
cd /Users/karina/Desktop/miniapp/digital-clinic

# Замени USERNAME на свой GitHub username
git remote set-url origin https://github.com/karinasekk-lab/digital-clinic.git
git branch -M main
git push -u origin main
```

- Перейди https://railway.app/dashboard

- **Deploy from GitHub repo**
- Find repo: `digital-clinic`

- Railway автоматически построит и задеплоит
- Жди ~2-3 мин, получишь live URL 🎉
